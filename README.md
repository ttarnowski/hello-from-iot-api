# 100DaysOfCodeInIoT Challenge Day 9
Go Serverless Hello From IoT webservice

## Requirements
- AWS Account
- AWS CLI
- Go

## Building
- Install serverless: `yarn install`
- Run `make build`

## Deploying
Run `make deploy`
